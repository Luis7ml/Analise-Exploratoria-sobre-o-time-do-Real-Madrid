# Análise Exploratória - Real Madrid

Este projeto realiza uma análise exploratória de dados dos jogadores do Real Madrid, utilizando Python e bibliotecas de análise de dados.

Objetivo
Explorar informações como idade, nacionalidade, posição e desempenho dos jogadores.

Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn

Estrutura do projeto
- Analise.ipynb → Notebook com a análise
- Tabela_Real_Madrid.csv → Base de dados

Como executar
1. Clone o repositório
2. Instale as dependências
3. Abra o notebook no Jupyter ou VS Code

Resultados
Foram gerados gráficos e estatísticas que ajudam a entender o perfil do elenco.
